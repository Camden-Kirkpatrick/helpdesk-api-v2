Helpdesk API

A simple helpdesk ticket system demonstrating a full-stack CRUD API built with FastAPI and SQLite.

Endpoints:

  • GET:
    • List all tickets: /api/tickets
    • Query for a ticket by id: /api/tickets/{ticket_id}
    • Query for tickets using query parameters:
      /api/tickets/search?title=<title>
      /api/tickets/search?priority=<1â€“5>
      /api/tickets/search?status=<open|closed|in_progress>
  • POST (via form): /api/tickets
  • PATCH (via form): /api/tickets/{ticket_id}
  • DELETE (via form): /api/tickets/{ticket_id}
